# Joseph Young Portfolio WordPress Theme

A modern, responsive WordPress theme for Joseph Young's portfolio website.

## Installation Instructions

### Step 1: Download and Install the Theme

1. Download the `portfolio-theme.zip` file
2. Log in to your WordPress admin dashboard
3. Go to **Appearance > Themes**
4. Click **Add New** > **Upload Theme**
5. Choose the `portfolio-theme.zip` file and click **Install Now**
6. Click **Activate** once installation is complete

### Step 2: Configure Theme Settings

1. Go to **Appearance > Customize**
2. Configure the **Hero Section**:
   - Set your name (default: Joseph Young)
   - Add your subtitle and description
   - Add your GitHub, LinkedIn, and Email links
3. Configure the **About Section**:
   - Upload your profile image
   - Add your about text
4. Click **Publish** to save your changes

### Step 3: Add Your Projects

1. In your WordPress dashboard, go to **Projects > Add New**
2. Add a project title
3. Add project description in the content editor
4. Set a **Featured Image** (this will be the project thumbnail)
5. In the **Project Details** section:
   - Add technologies/tags (comma-separated, e.g., "Python, React, Node.js")
   - Add GitHub link (optional)
   - Add Demo link (optional)
6. Click **Publish**

### Step 4: Customize Skills (Optional)

The skills section is hardcoded in the theme. To customize it:

1. Edit the file: `wp-content/themes/portfolio-theme/index.php`
2. Find the `#skills` section (around line 100)
3. Modify the skill categories and items as needed

### Step 5: Test Contact Form

The contact form will send emails to your WordPress admin email address (set in **Settings > General**).

## Features

- **Responsive Design**: Works perfectly on all devices
- **Custom Post Type**: Manage projects easily from WordPress admin
- **Theme Customizer**: Edit content without touching code
- **Contact Form**: Built-in contact form with email notifications
- **Smooth Scrolling**: Professional navigation experience
- **Modern Design**: Clean, professional aesthetic matching the original design

## Default Content

The theme comes with default content for:
- Hero section with your information
- Four skill categories (Programming, Engineering, Robotics, Tools)
- About section
- Contact form

## Recommended Plugins

While not required, these plugins enhance functionality:
- **Contact Form 7** (if you prefer a plugin-based contact form)
- **Yoast SEO** (for better SEO)
- **WP Super Cache** (for faster loading)
